Basic scene with hotspots

          
Every scene needs text:. Add hotspots: with a list of clickable areas.

          
text: You enter the hallway.
img: media/hallway.jpg

hotspots:
  - label: Open the red door
    region: [12, 20, 25, 60]
    next:
      text: Behind the red door is a kitchen.
      img: media/kitchen.jpg
      hotspots:
        - label: Open the fridge
          region: [60, 15, 25, 70]
          next:
            text: Empty. You starve. Game over.
            dead: true

        


        

          
Region format

          
region: [x, y, width, height] — all values are percentages of the image size. Use the Hotspot Drawer in column 2 to generate these automatically.

          
region: [10, 25, 40, 50]
# x=10%  y=25%  w=40%  h=50%

        


        

          
Dead end & win

          
Add dead: true or win: true to a scene — no hotspots needed. The text: becomes the outcome message.

          
next:
  text: You escape through the skylight!
  win: true

next:
  text: The door was locked. Game over.
  dead: true

        


        

          
Hotspot Drawer workflow

          

            1. Drop your image into the middle column

            2. Type a label in the label box

            3. Click & drag to draw a rectangle on the image

            4. Repeat for each hotspot

            5. Click Copy YAML to copy the hotspot block

            6. Paste it into the correct scene in the YAML editor
          

        


        

          
Full workflow

          

            1. Edit maze.yaml (here or in any text editor)

            2. Click Download maze.js in the top bar

            3. Replace maze.js in your project folder

            4. Commit both files to GitHub

            5. Open game.html to play!
