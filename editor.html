Basic scene

Every scene needs a text: field. Add left: and/or right: to branch.

text: You enter the hallway.

left:
  text: You go left into the kitchen.

right:
  text: You go right into the lounge.

Dead end

Add dead: true — no branches allowed. The text: becomes the failure message.

left:
  text: You fall into a trap. Game over.
  dead: true

Win condition

Add win: true — no branches allowed. The text: becomes the success message.

right:
  text: You made it! Congratulations.
  win: true

Images

Drop images in the media/ folder. Reference them by filename.

text: You enter the studio.
img: media/studio.jpg
leftImg: media/couch.jpg
rightImg: media/hallway.jpg

Disable one button

Simply omit that branch — the button will appear greyed out.

text: Dead end corridor — only one way out.
right:
  text: You escape through the window.

The workflow

1. Edit maze.yaml (here or in any text editor)
2. Click Download maze.js in the top bar
3. Drop the downloaded file into your project folder
4. Commit both maze.yaml and maze.js to GitHub
5. Open game.html to play!
